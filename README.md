# Advent of Code Solutions

Project structure

```
└── aocgleam # per language
    ├── src
    │   ├── inputs
    │   │   └── day1 # test data
    │   └── users # dir per user
    │       └── susliko
    └── test

```

## Gleam
To run a module (when inside [./aocgleam](./aocgleam)):
```
gleam run -m users/<user>/<modulename>
```
